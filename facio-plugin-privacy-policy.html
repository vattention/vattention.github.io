Facio Plugin Privacy Policy

Last Updated: December 6, 2024

1. Overview

Facio ("we", "our", or "the extension") is a product developed by Vattention. We are committed to protecting your privacy. This Privacy Policy explains how Facio collects, uses, and shares user data when you use our Chrome extension.

2. Data Collection

Facio collects the following types of data to provide its image and media collection functionality:

2.1 Website Content

When you use Facio to save media, we collect:

  • Images: Image files, URLs, and screenshots from web pages
  • Videos: Video files and URLs from web pages
  • Audio: Audio files and URLs from web pages
  • Screenshots: Captured screenshots of visible areas, selected areas, or full pages

Collection Method: Data is collected only when you explicitly trigger a save action (right-click menu, keyboard shortcut, drag-and-drop, or extension popup).

2.2 Web History

When you save media, we collect the following metadata associated with the saved content:

  • Page Titles: The title of the webpage where media was saved (document.title)
  • Page URLs: The URL of the webpage where media was saved (window.location.href)
  • Source URLs: The original URLs of saved images, videos, or audio files

Important: We do NOT collect a comprehensive browsing history. We only collect page titles and URLs for pages where you explicitly save media, and this metadata is stored only with the saved media for organizational purposes.

2.3 User Activity

We collect minimal user activity data:

  • Widget Positions: The position of the drag-and-drop float widget on your screen (stored locally via Chrome's storage API)

This data is used solely to remember your UI preferences and provide a consistent user experience.

2.4 Data We Do NOT Collect

Facio does NOT collect:

  • Personally identifiable information (name, address, email, age, identification numbers)
  • Health information
  • Financial and payment information
  • Authentication information (passwords, credentials, PINs)
  • Personal communications (emails, texts, chat messages)
  • Location data (GPS coordinates, IP addresses for location tracking)
  • Comprehensive browsing history (only page titles/URLs for saved media)

3. How We Use Your Data

All collected data is used solely for the extension's single purpose: facilitating the collection and organization of images and media from web pages.

3.1 Website Content

  • To save images, videos, audio, and screenshots to your local device
  • To organize saved media with associated metadata
  • To display saved media in the Electron application

3.2 Web History (Page Titles/URLs)

  • To associate saved media with its source webpage for organizational purposes
  • To help you identify where saved media came from

3.3 User Activity (Widget Positions)

  • To remember your UI preferences and restore widget positions across browser sessions

We do NOT use your data for:

  • Advertising or marketing
  • Analytics or tracking
  • Determining creditworthiness or lending purposes
  • Any purpose unrelated to media collection and organization

4. How We Share Your Data

Facio does NOT sell, rent, or transfer your data to third parties.

4.1 Local Electron Application

Saved media and associated metadata (images, videos, audio, screenshots, page titles, and URLs) are sent to a local Electron application running on your device at localhost:3000 or localhost:51678. This is a local service running on your computer, not a remote server.

Purpose: To organize and store your saved media in the Electron application's local database.

Data Shared: Only the media files and metadata you explicitly choose to save.

4.2 Local Storage

Widget position preferences are stored locally using Chrome's chrome.storage.local API. This data never leaves your device.

4.3 No Third-Party Sharing

We do NOT share your data with:

  • External servers or cloud services
  • Analytics services (Google Analytics, etc.)
  • Advertising networks
  • Social media platforms
  • Data brokers or third-party services
  • Any other external parties

5. Data Storage and Security

5.1 Local Storage Only

All data processed by Facio stays on your device:

  • Media Files: Saved images, videos, audio, and screenshots are stored locally in the Electron application's database
  • Metadata: Page titles, URLs, and source information are stored locally with saved media
  • Preferences: Widget positions and UI settings are stored in Chrome's local storage

5.2 No Cloud Storage

We do not use any cloud storage or remote servers for user data. All data remains on your local device.

5.3 Data Security

Since all data is stored locally on your device:

  • Your data is as secure as your local computer
  • We recommend keeping your operating system and Chrome browser updated
  • Use strong passwords and enable disk encryption if needed

6. Permissions Explanation

Facio requests the following permissions to function properly:

activeTab

Purpose: Capture screenshots of the current tab's visible content.

Usage: Only activated when you explicitly trigger a capture action via keyboard shortcuts or the extension popup.

Data Collected: Screenshots (only when you request them).

tabs

Purpose: Query active tab metadata and communicate with content scripts.

Usage: To get page titles and URLs for saved media, and to coordinate image detection operations.

Data Collected: Page titles and URLs (only for pages where you save media).

storage

Purpose: Store user preferences locally.

Usage: To remember widget positions and UI preferences.

Data Collected: Widget position coordinates (stored locally only).

contextMenus

Purpose: Provide right-click menu options for quick media saving.

Usage: Displays "Save to Facio" options in context menus.

Data Collected: Only the media element you right-click on.

downloads

Purpose: Reserved for future functionality (currently not used).

Usage: The extension currently sends data to the Electron app instead of using the downloads API.

notifications

Purpose: Show success/error notifications.

Usage: Displays temporary notifications for user feedback.

Data Collected: None (notifications are one-way communication).

host_permissions (<all_urls>)

Purpose: Access images and media from any website.

Usage: Required to detect and collect images from different domains.

Data Collected: Only image/video/audio URLs and metadata from pages where you save media. No personal data, form inputs, or other sensitive information is accessed.

7. User Rights and Control

7.1 Access to Your Data

All your saved media and metadata are stored locally in the Electron application. You can access, view, and manage all your data directly through the Electron app interface.

7.2 Deletion of Data

You can delete your data at any time by:

  • Deleting saved media through the Electron application
  • Uninstalling the Chrome extension (this removes all local storage preferences)
  • Clearing Chrome's local storage for the extension

7.3 Data Portability

Since all data is stored locally on your device, you have full control and can export or backup your data through the Electron application.

8. Children's Privacy

Facio does not knowingly collect data from anyone, including children under 13. The extension does not require user registration or collect personal information, making it safe for all ages. However, we recommend that children under 13 use this extension with parental supervision.
