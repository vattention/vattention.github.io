Product Engineer

Job Description

产品工程师负责将我们的AI技术转化为直观、强大的用户界面和体验。 您将直接与用户合作，理解他们的需求，并构建他们喜爱的产品功能。

主要职责：

  • 设计和实现代码编辑器的核心功能
  • 构建智能代码建议和自动完成系统
  • 开发用户友好的AI交互界面
  • 优化产品性能和用户体验
  • 集成新的AI功能到现有产品中

任职要求：

  • 具备学士或以上学位，不限定专业，理/工/农/医/文/史/哲/法/管/艺等均可
  • 熟练使用 Cursor / Claude Code / Augment 等任意 AI Coding Agent 进行产品原型设计
  • 对以下任意领域之一有深入的了解：视频后期制作、3D 建模、动画制作、平面设计、UI/UX 设计
  • 自驱力高于 80% 同龄人，动手能力强
  • 能从帮助团队伙伴的过程中获得快乐
  • 乐于表达自己的真实想法，不内耗

About Vattention

我们是一家专注于创新的科技公司，致力于自动化编程。 我们旅程的第一步是构建专业程序员的最佳工具，结合创新研究、设计和工程。

我们是一群工程师和科学家，他们构建了受人喜爱的产品，创建了大型开源项目， 创办了规模化企业，赢得了奥林匹克奖牌，并发表了重要研究。

我们的组织非常扁平，团队小而精干。我们特别喜欢那些寻求真理、充满激情和富有创造力的人。 我们享受激烈的辩论、疯狂的想法和代码发布。
