AI 工程师，后端与智能体

职位描述

负责 AI 核心产品的后端分布式架构设计与开发，以及 Agent 产品端侧后台的引擎层功能开发。

主要职责：

  • 负责AI核心产品的后端架构设计与开发，构建高可用、低延迟的后端系统
  • 负责视频 AI Agent 功能开发，包括决策、调度、记忆、意图理解、虚拟沙箱等
  • 负责在云平台开发各项服务，基础设施和架构搭建与优化，基于 Docker、K8s 等搭建完善的CI/CD流水线、结构化日志、分布式追踪及性能监控体系，保障系统可观测性与故障快速排查能力
  • 与产品、客户端及算法团队紧密协作，明确技术需求与实现路径，共同推进 AI 产品从概念到落地的全流程
  • 以强烈的 ownership 驱动产研团队项目推进、公司核心团队建设

任职要求：

  • 计算机科学、信息技术、软件工程等相关专业学士或以上学历，扎实的计算机基础
  • 3 年以上后端开发经验，1 年以上 AI 相关经验
  • 熟悉 Python / FastAPI 等常见框架，具备扎实的后端开发与分布式系统经验，熟练掌握 MySQL / PostgreSQL / Redis / Kafka 等常见基础服务
  • 具备AI技术工程化落地经验，熟悉AI Agent、Workflow、RAG等核心概念，熟悉 pgvector、Pinecone 等向量库的使用
  • 具备云原生与工程化实践能力：熟悉至少一个主流云平台（AWS、阿里云等），有云服务部署经验；熟悉Docker、K8s等容器技术，掌握CI/CD构建、日志监控和故障排查方法
  • 熟悉视频开发流程和AI在视频场景中的应用场景（如智能剪辑、内容生成、字幕生成等）者优先
  • 具有开放心态与技术攻坚能力，愿意主动学习新技术，能快速适应业务变化并解决复杂技术挑战
  • 自驱力高于80%同龄人，动手能力强
  • 能从帮助团队伙伴的过程中获得快乐
  • 乐于表达自己的真实想法，不内耗
