Engineering Full-time ⚲ Hangzhou / Beijing / Remote

Software Engineer, Desktop Video App

Job Description

桌面客户端工程师负责开发和维护跨平台的桌面客户端应用程序，确保高性能、稳定性和优秀的用户体验。

主要职责：

  • Design and develop cross-platform video post-production desktop clients for Windows and macOS
  • Build UI and interaction logic with Qt, design client/server communication, integrate AI-native capabilities to support intelligent post-production features
  • Lead performance optimization, bug fixing, iterative upgrades, and cross-platform adaptation

任职要求：

  • Bachelor's degree or above, 3+ years desktop development experience, solid C/C++ and algorithm foundation
  • Proficient with Qt framework; experienced in multithreading, network programming, and desktop cross-platform adaptation
  • Familiar with memory leak and crash troubleshooting; skilled with mainstream debugging tools
  • Understanding of basic AI capabilities; experience integrating AI features on desktop is a plus
  • Familiarity with audio-video processing, or experience using/developing video editing software is a plus
  • Self-driven with stronger initiative than 80% of peers; highly hands-on
  • Willing to trust teammates first; comfortable expressing genuine ideas, without overthinking
  • Find joy in helping and supporting teammates
