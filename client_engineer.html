Software Engineer, Desktop Client

Job Description

桌面客户端工程师负责Windows/macOS跨平台视频编辑工具客户端的架构设计、核心开发与功能实现，基于QT框架开发界面与交互逻辑，支撑视频编辑核心场景需求。

Key Responsibilities:

  • Responsible for architecture design, core development and feature implementation of Windows/macOS cross-platform video editing tool clients
  • Develop UI and interaction logic based on QT framework (Qt Widgets/QML), supporting core video editing scenario requirements
  • Design client-server communication solutions, collaborating with product, backend and design teams to advance interaction implementation
  • Participate in AI feature integration, interface with AI APIs and implement local adaptation, supporting AI-empowered scenarios like intelligent editing
  • Control product quality, lead performance optimization, issue fixing and iterative upgrades, solving technical challenges like cross-platform adaptation and concurrency

Qualifications:

  • Bachelor's degree or above, 3+ years of desktop development experience, solid C/C++ foundation and algorithmic ability
  • Proficient in QT framework development, deep mastery of Qt Widgets or QML/Qt Quick technology stack
  • Skilled in multi-threading, network programming (TCP/IP), with cross-platform (Windows/macOS) development adaptation experience
  • Familiar with memory leak and crash troubleshooting methods, proficient in using mainstream debugging tools
  • Understanding of basic AI functionality principles, experience in desktop AI feature integration is preferred
  • Strong self-motivation, excellent cross-team collaboration skills
  • Familiar with audio-video processing technology, or experience in video editing software usage/development is preferred
